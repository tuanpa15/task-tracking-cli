# Task Tracking CLI

A simple command-line interface (CLI) application for tracking tasks.

## Features

- Add new tasks
- Update existing tasks
- Delete tasks
- List all tasks
- Filter tasks by status

## Installation

1. Clone the repository:

   ```sh
   git clone <repository-url>
   cd task-tracking-cli
   ```

2. Install dependencies:

   ```sh
   npm install
   ```

## Usage

Run the application:

```sh
node index.js
```
